# GuardRail Backend

## Railway par Deploy karne ka tarika:

1. GitHub par naya repo banao "guardrail-backend"
2. Ye saari files upload karo
3. Railway.app par jao
4. "New Project" → "Deploy from GitHub" → ye repo select karo
5. Environment Variables mein ye add karo:
   - `GEMINI_API_KEY` = tumhari Gemini API key
   - `PORT` = 3000
6. Deploy!
